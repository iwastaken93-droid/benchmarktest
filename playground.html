CORS Troubleshooting Helper

Browser security blocks cloud requests directly from static HTML files. Choose one solution:

Option A: Browser Extension (Easiest)

Install the **Allow CORS** extension in Chrome/Firefox and toggle it on.

Option B: Local CORS Proxy

Start a local node proxy in your command line:

npx cors-anywhere

Then set CORS Proxy input field to: http://localhost:8080/

Option C: Local NIM Container

If self-hosting a NIM container, start it with CORS origins enabled:

-e NIM_CORS_ALLOW_ORIGINS="*"
